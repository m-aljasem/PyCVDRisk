# Quick Start Guide

## Prerequisites
- Node.js 18+ installed
- npm or yarn

## Setup

```bash
# Navigate to website directory
cd website

# Install dependencies
npm install

# Run development server
npm run dev
```

Visit http://localhost:3000

## Deploy to Vercel

1. Push code to GitHub
2. Go to vercel.com
3. Import your repository
4. Set root directory to `website`
5. Deploy!
